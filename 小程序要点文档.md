1。每个小程序的布局前都会自动的添加一个page，设置高度为100%，即可
2.搜索框禁用disabled="disabled"
3.icon 居中可以使用line-height
4.data-  自定义属性   e.currentTarget.dataset.current   可以把设置的自定义内容打印出来
5选项卡可以使用swiper
6.node中静态资源的托管可以使用static
7.css中设置2行，超出部分...
8.搜索框的首行缩进需要用刀text-indent
9.wx:key="property" 其中property是代表在 for 循环的 array 中 item 的某个 property，
该 property 的值需要是列表中唯一的字符串或数字，且不能动态改变。类似于字典的key值
10.css的span元素设置margin是无效的，必须转换为inline-block
11.为列表的每个小项都添加点击事件，并获得每个项的一些数据。
<view bindtap='click' data-name='{{item.name}}'>{{item.name}}</view>
var name = e.currentTarget.dataset.name;
12.Array.prototype.filter的返回的项必须是具体的一项
13.小程序监听数据变化 引入watch库  https://github.com/jayZOU/watch
